## Installation and Setup Instructions

You will need `node` and `npm` or `yarn` installed globally on your machine to download the librarys an start the app.

Installation:

`npm install`  
  
or  
  
`yarn install`

To Start Server:

`npm start`  

To Visit App:

`127.0.0.1:3000`
